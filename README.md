# Zypher API

## Methods

- `GET` - `/user`
- `GET` - `/event`
- `POST` - `/user` - body => `{"email": String}`
- `POST` - `/event` - body => `{"eventName": String, "eventDate": "YYYY-MM-DD"}`
- `POST` - `/user/add` - body => `{"eventId": ObjectId, userId: "userId"}`

## Run Program

Go to the directory path inside your terminal and run

- `npm install`
- `npm start`
